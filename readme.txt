Etapas do de instalação do projeto:

1. Instale o composer no servidor apache;
2. Nas variáveis de ambiente do windos, na variável "PATH", inclua um novo caminho indicando a pasta onde se encontra o bat do composer, arquivo composer.bat;
3. Abra o prompt e acesse a pasta base do seu projeto e execute o comando "composer require firebase/php-jwt" para instalação da biblioteca JWT;
4. No PHP crie um VirtualHost chamado "ordemservico";
